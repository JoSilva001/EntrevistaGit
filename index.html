Descrição Comando
Ver a versão git --version
Criar a pasta mkdir nome da pasta
Inicializar o repositório git init
visualizar informações git status
Listar recurso da pasta ls
Navegar nas pastas cd ou cd..
Abrir nano nano nome do arquivo
fechar nano CTRL + X
Quem é você? git config --local user.name "seu nome" e user.email "seu email"
Listar os commits git log ou git log --oneline ou git log -p ; obs: use q para sair

listar repositório remoto git remote

Enviar para o remoto com -u git push -u origin master, depois use o git push
Desfazer um commit git revert coloque o hash do seu commit
